# L031.5 --- Algorithmes TurfIA --- Détection des Value Bets

## 1. Objectif

Ce document formalise la méthode d'identification des *Value Bets* dans
TurfIA.

Un Value Bet correspond à un cheval dont la probabilité estimée par
TurfIA est supérieure à la probabilité implicite de sa cote.

------------------------------------------------------------------------

## 2. Définition

Soient :

-   `P_turfia` : probabilité estimée par TurfIA ;
-   `P_marche` : probabilité implicite du marché.

La probabilité implicite est calculée par :

``` text
P_marche = 1 / Cote
```

Après correction éventuelle de la marge opérateur.

------------------------------------------------------------------------

## 3. Valeur

La valeur est définie par :

``` text
Valeur = P_turfia - P_marche
```

Ou, en ratio :

``` text
Ratio = P_turfia / P_marche
```

Un ratio supérieur à 1 indique une sous-évaluation potentielle par le
marché.

------------------------------------------------------------------------

## 4. Critères d'éligibilité

Un cheval peut être qualifié de Value Bet si les conditions suivantes
sont réunies :

-   Score TurfIA supérieur au seuil minimal ;
-   probabilité TurfIA \> probabilité implicite ;
-   risque inférieur au seuil maximal ;
-   données jugées fiables ;
-   absence d'anomalie sur les cotes.

Toutes les conditions sont paramétrables.

------------------------------------------------------------------------

## 5. Niveaux de Value

  Ratio         Qualification
  ------------- ---------------
  \< 1,00       Aucune valeur
  1,00 à 1,10   Faible
  1,10 à 1,25   Intéressante
  \> 1,25       Forte

Les seuils sont ajustables dans les paramètres métier.

------------------------------------------------------------------------

## 6. Pondération

Le score Value Bet peut être modulé selon :

-   stabilité des cotes ;
-   consensus presse ;
-   historique TurfIA ;
-   réussite jockey ;
-   réussite entraîneur ;
-   aptitude au parcours.

La valeur finale reste comprise entre 0 et 100.

------------------------------------------------------------------------

## 7. Intégration au moteur

Le score Value Bet intervient :

-   dans le classement final ;
-   dans la sélection des outsiders ;
-   dans le choix des paris ;
-   dans l'estimation du ROI théorique.

Il ne remplace jamais le Score TurfIA.

------------------------------------------------------------------------

## 8. Pseudo-code

``` text
pour chaque partant

    calculer P_turfia

    calculer P_marche

    calculer valeur

    si valeur > seuil

        marquer Value Bet

    sinon

        ignorer
```

------------------------------------------------------------------------

## 9. Traçabilité

Chaque Value Bet conserve :

-   cote utilisée ;
-   probabilité implicite ;
-   probabilité TurfIA ;
-   valeur calculée ;
-   paramètres ;
-   version du modèle.

------------------------------------------------------------------------

## 10. Validation

La pertinence du détecteur est évaluée périodiquement à partir :

-   du ROI des Value Bets ;
-   du taux de réussite ;
-   du nombre de faux positifs ;
-   de la stabilité des performances.

Toute évolution est validée exclusivement sur les résultats historiques.
