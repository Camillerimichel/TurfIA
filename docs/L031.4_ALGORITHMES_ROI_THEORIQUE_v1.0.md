# L031.4 --- Algorithmes TurfIA --- Calcul du ROI théorique

## 1. Objectif

Le ROI théorique estime la rentabilité attendue d'une course avant son
départ.

Il constitue, avec le Score TurfIA et le niveau de risque, le troisième
indicateur de décision du moteur.

------------------------------------------------------------------------

## 2. Principe

Le ROI théorique repose sur la comparaison entre :

-   la probabilité estimée par TurfIA ;
-   la probabilité implicite des cotes ;
-   les rapports attendus selon les types de paris.

Le calcul est réalisé pour chaque cheval puis agrégé au niveau de la
course.

------------------------------------------------------------------------

## 3. Probabilité implicite

Pour une cote décimale :

``` text
Probabilité_implicite = 1 / Cote
```

Cette probabilité est ensuite corrigée afin de neutraliser la marge de
l'opérateur.

------------------------------------------------------------------------

## 4. Probabilité TurfIA

La probabilité TurfIA est dérivée du score normalisé du cheval.

Exemple de principe :

``` text
Probabilité_TurfIA =
Score_cheval
/
Somme(scores de tous les partants)
```

La fonction exacte de conversion pourra évoluer sans modifier
l'architecture.

------------------------------------------------------------------------

## 5. Espérance de gain

Pour chaque pari potentiel :

``` text
Espérance =
Probabilité_TurfIA × Rapport_estimé
-
Mise
```

Une espérance positive constitue une condition nécessaire à la
recommandation d'un pari.

------------------------------------------------------------------------

## 6. ROI théorique

Le ROI théorique est calculé par :

``` text
ROI =
(Espérance / Mise)
× 100
```

Exemple :

-   Mise : 10 €
-   Espérance : 12 €

``` text
ROI = (12 - 10) / 10 × 100 = 20 %
```

------------------------------------------------------------------------

## 7. Agrégation par course

Le ROI global de la course est obtenu par combinaison des ROI des
différents paris recommandés.

Les pondérations tiennent compte :

-   du type de pari ;
-   de la confiance ;
-   du risque.

------------------------------------------------------------------------

## 8. Influence sur la décision

La décision finale combine :

``` text
Score TurfIA

+

Risque

+

ROI théorique
```

Une course n'est retenue que si ces trois indicateurs sont cohérents.

------------------------------------------------------------------------

## 9. Seuils

Exemple de politique :

  ROI théorique   Interprétation
  --------------- ----------------
  \< 0 %          À éviter
  0 à 10 %        Faible intérêt
  10 à 25 %       Intéressant
  \> 25 %         Très favorable

Ces seuils sont entièrement paramétrables.

------------------------------------------------------------------------

## 10. Pseudo-code

``` text
pour chaque cheval

    calculer probabilité TurfIA

    calculer probabilité implicite

    calculer espérance

    calculer ROI

agréger les résultats

déterminer le ROI de la course
```

------------------------------------------------------------------------

## 11. Traçabilité

Pour chaque analyse sont conservés :

-   probabilité TurfIA ;
-   probabilité implicite ;
-   espérance ;
-   ROI individuel ;
-   ROI global ;
-   paramètres utilisés.

------------------------------------------------------------------------

## 12. Validation

Le modèle de calcul du ROI théorique est comparé régulièrement au ROI
réellement obtenu.

L'écart entre estimation et résultat observé constitue un indicateur
majeur de calibration du modèle et alimente le processus d'amélioration
continue.
