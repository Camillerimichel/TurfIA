# L029 --- Glossaire v1.0

## 1. Objectif

Ce glossaire définit la terminologie officielle utilisée dans la
documentation de TurfIA.

## 2. Définitions

  -----------------------------------------------------------------------
  Terme                        Définition
  ---------------------------- ------------------------------------------
  Analyse                      Exécution du moteur TurfIA sur une course.

  Analyse finale               Analyse recalculée avant le départ
                               officiel.

  Base                         Cheval présentant la meilleure probabilité
                               de réussite.

  Chance régulière             Cheval présentant une forte probabilité
                               d'être à l'arrivée.

  Outsider                     Cheval peu joué mais jugé compétitif.

  Tocard                       Cheval spéculatif à forte cote.

  Course                       Épreuve hippique analysée.

  Réunion                      Ensemble des courses d'un même hippodrome.

  Partant                      Cheval déclaré au départ.

  Score TurfIA                 Indicateur de confiance compris entre 0 et
                               100.

  ROI                          Retour sur investissement.

  Value Bet                    Cheval dont la probabilité TurfIA dépasse
                               la probabilité implicite de la cote.

  Consensus presse             Niveau d'accord entre les pronostics de la
                               presse spécialisée.

  Pré-analyse                  Première analyse réalisée avant la course.

  Contrôle ROI                 Comparaison entre les prévisions et les
                               résultats réels.
  -----------------------------------------------------------------------

## 3. Abréviations

  Abréviation   Signification
  ------------- ------------------------------------------------
  API           Application Programming Interface
  CI/CD         Continuous Integration / Continuous Deployment
  JSON          JavaScript Object Notation
  REST          Representational State Transfer
  ROI           Return On Investment
  SQL           Structured Query Language
  VPS           Virtual Private Server

## 4. Conclusion

Ce glossaire constitue le vocabulaire de référence de l'ensemble de la
documentation et du développement de TurfIA.
