## Objectif

TurfIA est un système d'analyse quantitative des courses hippiques visant à améliorer durablement le retour sur investissement (ROI) grâce à des analyses reproductibles fondées sur des données.

Le projet évalue d'abord l'intérêt de jouer une course avant de sélectionner les chevaux.

---

## Principes

- priorité au ROI long terme ;
- décisions fondées sur des données ;
- historique immuable ;
- amélioration continue validée ;
- séparation des données, traitements et présentation.

---

## Cycle

1. Pré-analyse
2. Analyse finale
3. Contrôle des résultats
4. Mise à jour de l'historique
5. Amélioration continue
