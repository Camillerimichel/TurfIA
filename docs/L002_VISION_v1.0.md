# TurfIA

## L002_VISION_v1.0

**Version :** 1.0
**Statut :** Validé

## Vision

TurfIA est une plateforme d'aide à la décision destinée aux paris hippiques. L'objectif n'est pas de prédire systématiquement le vainqueur d'une course mais d'identifier les situations présentant une espérance de gain positive sur le long terme.

## Objectifs stratégiques

- Décider de jouer ou non une course.
- Quantifier le niveau de confiance.
- Produire des sélections argumentées.
- Mesurer les performances réelles.
- Améliorer le modèle uniquement après validation statistique.

## Principes directeurs

- Reproductibilité.
- Traçabilité.
- Versionnement.
- Automatisation.
- Mesure permanente du ROI.

## Critères de réussite

Le succès du projet est évalué par la stabilité du ROI, la qualité des décisions et la conservation d'un historique complet permettant de comparer toutes les versions du modèle.
