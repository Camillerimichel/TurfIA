| Champ | Valeur |

## 1. Objet

Les tables statistiques consolident les résultats produits par TurfIA
afin de mesurer les performances du modèle dans le temps.

Toutes les données sont calculées automatiquement et ne sont jamais
modifiées manuellement.

------------------------------------------------------------------------

# 2. Principes

Les statistiques sont :

-   historisées ;
-   recalculables ;
-   indépendantes des analyses unitaires ;
-   utilisées pour l'amélioration continue.

------------------------------------------------------------------------

# 3. Vue d'ensemble

``` text
Analyses
    │
    ├── Statistique globale
    ├── Statistique par score
    ├── Statistique par hippodrome
    ├── Statistique par discipline
    ├── Statistique par pari
    └── Statistique par version du modèle
```

------------------------------------------------------------------------

# 4. Table `statistique_globale`

## Description

Synthèse générale de TurfIA.

  Colonne         Type            Contraintes   Description
  --------------- --------------- ------------- --------------------
  id              BIGINT          PK            Identifiant
  date_calcul     TIMESTAMP       NOT NULL      Date du calcul
  nb_courses      INTEGER         NOT NULL      Courses analysées
  nb_jouees       INTEGER                       Courses jouées
  nb_ignorees     INTEGER                       Courses non jouées
  mises           DECIMAL(12,2)                 Total des mises
  gains           DECIMAL(12,2)                 Total des gains
  profit          DECIMAL(12,2)                 Gains - mises
  roi             DECIMAL(8,2)                  ROI global
  taux_reussite   DECIMAL(5,2)                  \% de réussite

### Index

-   idx_stat_global_date

------------------------------------------------------------------------

# 5. Table `statistique_score`

## Description

Performances selon les tranches de confiance.

  Colonne         Type
  --------------- --------------
  id              BIGINT
  score_min       DECIMAL(5,2)
  score_max       DECIMAL(5,2)
  nb_courses      INTEGER
  nb_gagnantes    INTEGER
  roi             DECIMAL(8,2)
  taux_reussite   DECIMAL(5,2)

Exemple :

-   0--59
-   60--74
-   75--84
-   85--100

------------------------------------------------------------------------

  id              BIGINT

------------------------------------------------------------------------

# 7. Table `statistique_discipline`

  Colonne         Type
  --------------- ---------------
  id              BIGINT
  discipline_id   BIGINT
  nb_courses      INTEGER

------------------------------------------------------------------------

# 8. Table `statistique_pari`

## Description

Analyse des performances par type de pari.

  Colonne         Type
  --------------- ---------------
  id              BIGINT
  type_pari       VARCHAR(40)
  nb_paris        INTEGER
  mises           DECIMAL(12,2)
  gains           DECIMAL(12,2)
  profit          DECIMAL(12,2)
  roi             DECIMAL(8,2)
  taux_reussite   DECIMAL(5,2)

Valeurs possibles :

-   Simple Gagnant
-   Simple Placé
-   Couplé Gagnant
-   Couplé Placé
-   2 sur 4
-   Quinté Flexi

------------------------------------------------------------------------

# 9. Table `statistique_modele`

## Description

Comparaison des versions successives de TurfIA.

  Colonne          Type
  ---------------- --------------
  id               BIGINT
  version_modele   VARCHAR(20)
  date_debut       DATE
  date_fin         DATE
  nb_courses       INTEGER
  roi              DECIMAL(8,2)
  taux_reussite    DECIMAL(5,2)
  commentaire      TEXT

Cette table permet de valider objectivement toute évolution de
l'algorithme.

------------------------------------------------------------------------

# 10. Règles métier

-   les statistiques sont calculées automatiquement ;
-   aucun enregistrement n'est supprimé ;
-   les recalculs créent une nouvelle ligne ;
-   les résultats historiques sont figés.

------------------------------------------------------------------------

# 11. Historisation

Chaque calcul statistique est conservé.

Il est ainsi possible de reconstituer l'état des indicateurs à une date
donnée.

------------------------------------------------------------------------

# 12. Exploitation

Ces tables alimentent :

-   le tableau de bord principal ;
-   les graphiques d'évolution ;
-   les comparatifs de versions ;
-   les rapports quotidiens ;
-   le moteur d'amélioration continue.

Elles constituent la mémoire statistique de TurfIA.
