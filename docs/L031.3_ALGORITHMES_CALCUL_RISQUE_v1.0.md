# L031.3 --- Algorithmes TurfIA --- Calcul du risque

## 1. Objectif

Le risque mesure l'incertitude associée à une course indépendamment de
la qualité intrinsèque des chevaux.

Il complète le Score TurfIA afin de déterminer si une course doit être
jouée.

------------------------------------------------------------------------

## 2. Principe

Le risque est calculé à partir de plusieurs familles d'indicateurs.

``` text
Risque = Σ (Facteur × Pondération)
```

Le résultat est normalisé sur une échelle de 0 à 100 :

-   0 : risque minimal
-   100 : risque maximal

------------------------------------------------------------------------

## 3. Facteurs de risque

  Famille        Exemples
  -------------- ----------------------------------
  Marché         Forte volatilité des cotes
  Presse         Désaccord entre pronostiqueurs
  Course         Grand nombre de partants
  Terrain        Changement d'état de piste
  Historique     Faible reproductibilité
  Contexte       Nombre important de non-partants
  Statistiques   Manque de données fiables

Chaque facteur produit un sous-score.

------------------------------------------------------------------------

## 4. Normalisation

Chaque facteur est converti sur une échelle commune :

``` text
SousRisque = 100 × (Valeur - Min) / (Max - Min)
```

Les bornes sont configurables.

------------------------------------------------------------------------

## 5. Calcul global

Le risque global est obtenu par moyenne pondérée :

``` text
Risque_global =
Σ(SousRisque_i × Poids_i)
/ Σ(Poids_i)
```

Les pondérations sont stockées dans la table `parametre`.

------------------------------------------------------------------------

## 6. Interprétation

  Risque    Interprétation
  --------- ----------------
  0--25     Très faible
  26--50    Faible
  51--75    Moyen
  76--100   Élevé

------------------------------------------------------------------------

## 7. Impact sur la décision

Le risque est combiné au Score TurfIA.

Exemple de logique :

``` text
Score élevé + Risque faible
    ⇒ Jeu recommandé

Score élevé + Risque élevé
    ⇒ Réduction de mise

Score faible
    ⇒ Pas de jeu
```

Les seuils sont paramétrables.

------------------------------------------------------------------------

## 8. Influence sur la mise

Le risque peut réduire automatiquement la mise recommandée sans modifier
le classement des chevaux.

Cette règle permet de limiter la volatilité globale du portefeuille.

------------------------------------------------------------------------

## 9. Pseudo-code

``` text
calculer tous les facteurs

normaliser

pondérer

calculer le risque

combiner avec le score TurfIA

déterminer la décision finale
```

------------------------------------------------------------------------

## 10. Traçabilité

Chaque analyse conserve :

-   les sous-risques ;
-   les pondérations ;
-   le risque final ;
-   la décision associée.

Ces données alimentent les statistiques historiques.

------------------------------------------------------------------------

## 11. Validation

Toute modification de la formule de risque doit être validée sur
l'historique complet de TurfIA.

La nouvelle version n'est retenue que si elle améliore le ROI ajusté du
risque sans dégrader la stabilité du modèle.
