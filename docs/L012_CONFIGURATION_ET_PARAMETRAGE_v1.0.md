L012 — Configuration et paramétrage

## Objectif
Centraliser tous les paramètres configurables de TurfIA.

## Paramètres
- Pondérations des critères.
- Seuils de confiance.
- Budgets par niveau de confiance.
- Types de paris autorisés.
- Fréquences des traitements.
- Sources de données activées.
- Paramètres des connecteurs.

## Principes
- Aucun paramètre métier codé en dur.
- Versionnement de chaque configuration.
- Validation avant mise en production.
- Conservation de l'historique des modifications.
- Possibilité de restaurer une configuration antérieure.
